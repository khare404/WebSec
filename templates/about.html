About WebSec Scanner

WebSec Scanner is a lightweight tool designed for detecting common vulnerabilities in web applications. It aims to assist ethical hackers, students, and developers in understanding real-world security flaws and resolving them effectively.

Project Objectives:

  • Identify vulnerabilities in a web application with ease.
  • Support multiple modules with modular design.
  • Generate visually distinct output for easier analysis.
  • Encourage learning of secure coding and testing practices.

Vulnerability Types:

  • SQL Injection: Exploiting database queries through unsanitized input fields.
  • XSS: Injecting scripts that execute in users' browsers to steal data or hijack sessions.
  • Security Misconfig: Weak or default configurations that can expose critical data.
  • Directory Traversal: Accessing files outside the web directory using path traversal.
  • Enumeration: Discovering sensitive system data such as user accounts or services.

Note: This scanner is for ethical and educational use only. Do not scan targets without permission.
